# ARImport
## Getting Started
### Prerequisites
*   Go == 1.12.6
### Build and Run
```console
>> go build .\src\server\main.go
>> .\main.exe
```
### Packages
*   output
*   scheduler
#### Generate docs
##### HTTP :
```console
>> cd .\src\
>> godoc -http=:8002
```
*   Navigate to 'http://localhost:8002/pkg/github.com/ajagnic/ARImport/'
